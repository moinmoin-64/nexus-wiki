# SOFORT LÖSUNG - Passwort setzen

Du bist JETZT auf dem NixOS-System angemeldet als `nixos`.

## DIREKT EINGEBEN (auf deinem NixOS-Terminal):

```bash
sudo passwd nixos
```

Dann:
```
Neue UNIX-Passwort: nexus123
UNIX-Passwort wiederholen: nexus123
```
